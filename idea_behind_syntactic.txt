/*
    Recursive Descent Parser notes

    Each grammar rule is implemented as a function that returns 1 if the rule
    matches and 0 otherwise. Tokens are consumed with consume(), and nonterminals
    are consumed by calling their own functions.

    1. Left recursion elimination

    Direct left-recursive rules cannot be implemented directly because they
    would cause infinite recursion.

    Example:

        exprAdd: exprAdd (ADD | SUB) exprMul | exprMul

    was transformed into:

        exprAdd    ::= exprMul exprAddAux
        exprAddAux ::= (ADD | SUB) exprMul exprAddAux | epsilon

    The Aux functions return 1 at the end to represent epsilon.

    This was applied to:
        exprOr, exprAnd, exprEq, exprRel, exprAdd, exprMul, exprPostfix.

    2. Alternatives: a | b

    Alternatives are tested one by one, and the function returns 1 at the first
    match. If none match, it returns 0.

    Example:

        if (consume(INT)) return 1;
        if (consume(DOUBLE)) return 1;
        if (consume(CHAR)) return 1;
        if (consume(STRUCT)) {
            if (!consume(ID)) tkerr(crtTk, "missing ID after struct");
            return 1;
        }
        return 0;

    3. Optionality: a?

    Optional rules are called without requiring success.

        arrayDecl(); // optional

    This is used in rules like varDef, fnParam, and exprCast.

    4. Optional repetition: a*

    Repetitions are implemented using loops.

        while (1) {
            if (varDef()) {}
            else if (stm()) {}
            else break;
        }

    This matches rules like:

        stmCompound: LACC ( varDef | stm )* RACC

    5. Common prefix factorization

    Common prefixes are grouped before testing optional parts.

    Example:

        exprPrimary:
            ID ( LPAR ( expr ( COMMA expr )* )? RPAR )?
            | CT_INT | CT_REAL | CT_CHAR | CT_STRING | LPAR expr RPAR

    In code:

        if (consume(ID)) {
            if (consume(LPAR)) {
                ...
            }
            return 1;
        }

    This accepts:
        x
        f()
        f(a, b)
*/
